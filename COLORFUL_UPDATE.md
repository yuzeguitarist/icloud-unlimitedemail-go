## 🎨 配色原则

### 1. 语义化颜色
- **绿色系** - 成功、激活、完成
- **红色系** - 错误、删除、危险
- **黄色系** - 警告、停用、等待
- **蓝色系** - 信息、标签、处理中
- **品红系** - 标题、重要信息
- **青色系** - 提示、分隔线

### 2. 渐进式增强
- 普通操作使用基础颜色
- 重要信息使用亮色版本
- 错误/成功使用高对比度

### 3. 视觉层次
```
亮度级别：
  Level 1: 亮色 + 粗体 (标题、数字)
  Level 2: 亮色        (符号、强调)
  Level 3: 基础色      (正文、说明)
  Level 4: 灰色        (辅助、提示)
```

## 📊 颜色使用统计

- **绿色系**: 35% (成功状态、激活)
- **蓝色系**: 25% (信息、标签)

## 🔧 技术实现

### ANSI 颜色代码
```go
// 基础颜色
ColorRed     = "\033[31m"
ColorGreen   = "\033[32m"
ColorYellow  = "\033[33m"
ColorBlue    = "\033[34m"
ColorMagenta = "\033[35m"
ColorCyan    = "\033[36m"

// 亮色版本
ColorBrightRed     = "\033[91m"
ColorBrightGreen   = "\033[92m"
ColorBrightYellow  = "\033[93m"
ColorBrightBlue    = "\033[94m"
ColorBrightMagenta = "\033[95m"
ColorBrightCyan    = "\033[96m"

// 样式
ColorBold  = "\033[1m"
ColorReset = "\033[0m"
```

### 渐变进度条实现
```go
if progress < 0.3 {
    bar.WriteString(ColorBrightRed + "█" + ColorReset)
} else if progress < 0.7 {
    bar.WriteString(ColorBrightYellow + "█" + ColorReset)
} else {
    bar.WriteString(ColorBrightGreen + "█" + ColorReset)
}
```

### 彩虹动画实现
```go
colors := []string{
    ColorBrightCyan,
    ColorBrightBlue,
    ColorBrightMagenta,
    ColorBrightRed,
    ColorBrightYellow,
    ColorBrightGreen
}
colorIdx := idx % len(colors)
```
