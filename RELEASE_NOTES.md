# iCloud 隐藏邮箱管理工具 v2.0.0

## 🎉 重大更新

这是一个全新的版本，带来了完整的邮箱管理功能和美观的用户界面！

## ✨ 新功能

### 核心功能
- **完整的邮箱管理** - 创建、查看、停用、删除、重新激活
- **批量操作** - 支持批量创建和批量管理
- **智能重试** - 自动处理网络错误和API限流
- **实时进度** - 美观的进度条和状态显示

### 用户界面
- **彩色界面** - 使用ANSI颜色代码，提供美观的命令行体验
- **状态标识** - 用颜色区分不同的状态和操作类型
- **进度显示** - 实时显示操作进度和完成状态
- **交互式菜单** - 简洁明了的操作选择界面

### 技术改进
- **去除表情符号** - 使用颜色和文字标识替代表情符号
- **错误处理** - 更完善的错误处理和用户提示
- **配置管理** - 更灵活的配置文件结构
- **跨平台支持** - 支持 macOS、Linux、Windows

## 🚀 快速开始

1. **下载** 适合你系统的预编译版本
2. **解压** 下载的文件
3. **配置** 复制 `config.json.example` 为 `config.json` 并填入认证信息
4. **运行** 执行程序开始使用

## 📋 功能说明

### 主菜单选项

- **[1] 查看邮箱列表** - 显示所有已创建的隐藏邮箱及其状态
- **[2] 创建新邮箱** - 创建单个隐藏邮箱地址
- **[3] 停用邮箱** - 暂时停用邮箱（可恢复）
- **[4] 批量创建邮箱** - 一次性创建多个邮箱
- **[5] 彻底删除邮箱** - 永久删除已停用的邮箱
- **[6] 重新激活邮箱** - 恢复已停用的邮箱
- **[0] 退出** - 退出程序

### 颜色说明

- 🟢 **绿色** - 成功状态、激活状态
- 🔵 **蓝色** - 信息提示、进行中状态
- 🟡 **黄色** - 警告信息、已停用状态
- 🔴 **红色** - 错误信息、危险操作
- 🟣 **紫色** - 统计信息、计划信息
- 🔵 **青色** - 输入提示、标签信息

## ⚠️ 注意事项

1. **Cookie 时效性** - iCloud 认证 Cookie 会过期，需要定期更新
2. **请求频率** - 建议设置合理的延迟时间，避免被限流
3. **账号安全** - 不要分享包含认证信息的配置文件
4. **使用限制** - 请遵守 Apple 服务条款，合理使用
