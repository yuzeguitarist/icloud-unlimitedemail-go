# 发布说明

## v2.3.0 · 2025-11-3

### 🎉 新功能
- **邮箱保存功能**：自动保存生成的邮箱到文件，支持时间戳记录
- **开发者模式**：可选的调试功能，包含评分算法测试
- **程序设置重构**：统一的设置管理界面

### 🔧 改进
- **配置重载增强**：添加重试机制和错误处理，最多重试3次后安全退出
- **界面优化**：配置重载后自动清屏并重新显示主菜单
- **符号优化**：修复不兼容符号显示问题，保留优雅符号
- **菜单重构**：选项8改为"程序设置"，包含邮箱质量、保存设置、开发者模式

### 🐛 修复
- 修复配置重载失败时程序未真正退出的问题
- 修复配置重载后界面信息混乱的问题
- 优化配置文件默认值设置

---

## v2.2.0 · 2025-11-3

### 🎉 新功能
- **智能邮箱评分系统**：基于前缀结构、长度、可读性、安全性的多维度评分
- **配置热重载**：运行时自动检测配置文件变化并重新加载
- **进程安全管理**：防止多实例运行，支持优雅退出
- **网络重试机制**：自动重试失败的网络请求
- **邮箱质量设置**：可配置的评分权重和自动选择策略

### 🔧 改进
- **终端界面优化**：更好的颜色搭配和进度显示
- **错误处理增强**：更详细的错误信息和处理建议
- **配置管理优化**：支持配置验证和默认值设置

## v2.1.0 · 2025-11-3

### 🎉 新功能
- **批量创建邮箱**：支持一次性创建多个隐藏邮箱

- **进度显示**：添加彩虹进度条和 Spinner 动画
- **错误提示优化**：更友好的错误信息显示
